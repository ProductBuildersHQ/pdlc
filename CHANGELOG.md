## [v0.1.0] - 2026-07-20 ([`0abb1d3`](https://github.com/ProductBuildersHQ/pdlc/commit/0abb1d3))

### Highlights

- Initial release of the ProductBuildersHQ Product Development Lifecycle (PDLC): the normative specification plus a Go module that classifies a project repository against the layout contract and evaluates readiness for handoff to a builder
- Readiness evaluation emits a structured-evaluation report scoring each in-scope domain on artifact presence and, where a tool has run, quality — with unrun tools held at conditional rather than passing silently

### Added

- Layout contract as Go types with an embedded layout.yaml: canonical paths, per-profile requirement levels, authority, and detection heuristics for every product-definition artifact ([`b83e9eb`](https://github.com/ProductBuildersHQ/pdlc/commit/b83e9eb))
- Repository classifier that skips immovable (tool-native) and ignored (templates, examples, fixtures) directories, flags path conflicts, and never auto-proposes moves from content-only matches ([`b83e9eb`](https://github.com/ProductBuildersHQ/pdlc/commit/b83e9eb))
- pdlc.yaml project manifest with profile selection and per-domain override resolution ([`d66021b`](https://github.com/ProductBuildersHQ/pdlc/commit/d66021b))
- Readiness evaluator producing a structured-evaluation report: checklist-scale presence, leaf-report quality, and TOOL_NOT_RUN semantics for unrun tools ([`9694cf4`](https://github.com/ProductBuildersHQ/pdlc/commit/9694cf4))
- pdlc CLI (Cobra) with check, inventory, init, and layout commands ([`bdc8024`](https://github.com/ProductBuildersHQ/pdlc/commit/bdc8024))
- Go-first JSON Schema generation for the layout and project manifests, embedded for programmatic access ([`82ed0b0`](https://github.com/ProductBuildersHQ/pdlc/commit/82ed0b0))
- PDLC specification: lifecycle stages, canonical project layout, review-site assembly, quality evaluation, requirement registry and executable acceptance tests, agent-driven adoption, and the Product Baseline handoff contract ([`0abb1d3`](https://github.com/ProductBuildersHQ/pdlc/commit/0abb1d3))
